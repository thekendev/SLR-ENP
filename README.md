# SLR-ENP
A simple linear regression end to end ML project



### Project workflows

- Update config.yaml
- Update schema.yaml
- Update params.yaml
- Update the entity
- Update the configuration manager in src config
- Update the components
- Update the pipeline
- Update the main.py
- Update the app.py


# How to run?

### STEPS:

``` conda create -n mlproj python=3.8 -y ```


``` conda activate venv/ ```


``` pip install -r requirements.txt ```

``` python main.py ```

``` python app.py ```


``` Now open up your local host 0.0.0.0:8080 ```
